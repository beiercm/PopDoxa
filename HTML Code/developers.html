PopDoxa was a Senior Design Project for the year 2015.
The sponsor for the project was Professor Heinrich.
Below are the following developers that designed PopDoxa.

About the Developers

Back-End Developer

Chris worked on the SQL and PHP development. He was able to generate all of the SQL database that were necessary for the project and most of the php scripts. When the group was testing the site, Chris coded Python scripts that took an article and was able to create forums based on this article. Chris's work can be seen throughout the site in every page.

Mobile App Developer: Andriod

Philip was the mobile developer for the project. He studied and compiled the mobile application for PopDoxa in two semesters. With a few lessons from YouTube, Philip was able to become an accomplish mobile developer and design the app for the project. When there was problems with the code, he was able to figure it out without much difficulty and continue on with the coding process.

Front-End Developer

James was a font-end developer for PopDoxa. He focused on not only the designs of the site but also on a few webservice calls. His work can be seen in the log-in/log-out screen, register page, profile page, and poll's results. Through this Senior Design project, James was able to learn different aspects of web designing and some back-end developing.

Front-End Developer

Stacy was one of the two font-end developers for PopDoxa. She worked with Bootstrap and few other JavaScript plugins for the webpages. She collaborated with James and Chris to develop and design the site.
